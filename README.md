> Merge the contents of one page into another in Logseq.

## âœ¨ Features
* **Merge pages:** Move all blocks from one page into another page of your choice.
* **Search-based picker:** Quickly find the target page using the built-in search overlay.
* **Auto-navigation:** After merging, you are automatically redirected to the target page.

## ğŸ›  Usage
1. Navigate to the page you want to merge **from**.
2. Right-click the page title to open the context menu.
3. Select **Merge Page With**.
4. A search overlay will appear â€” type at least 1 characters to filter pages.
5. Click the target page to merge into. All blocks from the current page will be appended to the target page.
6. Press `Escape` or click outside the overlay to cancel.
